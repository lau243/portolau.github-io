Data Cleaning in SQL

Background

This project is mainly made for this portfolio, to demonstrate understanding of SQL language using MySQL. This particular dashboard inspired by youtube sources from AlexTheAnalyst with some spin on my own. Please visit my github repository to view the code.

Objective

The project based on open source Covid-19 data obtained from Our World in Data. The data contains daily cases, deaths, and vaccination numbers accross 200+ countries and the relevant statistics to draw correlation from. The goal is to write query in preparation for visualization, aspiring from WHO's Covid-19 Dashboard.

Method and Result

The data is reduced by dropping some redundant columns via Excel to reduce the loading burden. After the table are loaded locally, the table are normalized and columns changed to their appropriate typing. Then the relevant information are extracted with the aid of CTE, and views.
