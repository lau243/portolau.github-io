Product Reviews Project using NLP

Background

This project was made as final project for Data Analytics Independent Studies program by Kampus Merdeka in collaboration with GreatEdu. Please visit my github repository to view the code.

Objective

The project based on open source data obtained by Griko Nibras of Lazada Indonesia consumer electronic product sales from 2014 to 2019. Our objective alongside visualize the data, was to evaluate the effect of words used by sellers in their product name and customers in their reviews on product rating.

Method

The two prominent text features, productName and reviews, were used to generate a topic model that grouped similar worded products and reviews into one topic. Product name topics provided a way to separate products that belong to same category. Turns out that there were a lot of products that listed wrong categories, just in hope to increase impression. Both topic models used to feed into classification model, with the goal to identify products that result in high rating.

We used Light Gradient Boosting Machine as our primary algorithm considering its fast runtime and decent performance. The model were optimized for Kappa metric, as this results in the model that reduces the false positive rate, in this case the rate that poor rated products grouped into the high rating.

The whole operation run on python with the preparation done in pandas and numpy package, with the machine learning done in pycaret.
