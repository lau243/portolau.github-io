Stats Course Lab in R

Background

This is a compilation of statistics courses lab that i've done in uni using R software. Courses mentioned are Linear Models and Survival Analysis.

Linear Models

The course is about linear regression method along with its assumptions, implication, and measure of performance. This project is a final assignment to said course as a team. Our choice being a dataset of countries with its characteristics, especially its life expectancy. The model goal is to use the other field to predict a hypothetical countries life expectancy. Please visit my github repository linked below to view the code.

Survival Analysis

The course is about statistical analysis of survival data. The usual context for survival data is time of death of patients after a certain treatment. Using parametric or nonparametric approach, we can estimate the cumulative risk of death and survival rate of patients based on certain traits or behaviour. R provides a survival library to easily process survival data and make estimates to said measures. Please visit my github repository linked below to view the code.
